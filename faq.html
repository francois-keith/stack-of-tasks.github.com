Workaround

The issue has been signaled in the github page of the package sot-dynamic but the solution for this problem has not be found yet. A workaround consists in loading an element of sot-dynamic prior to any load of an element of sot-core: 

from dynamic_graph.sot.dynamics import AngleEstimator
from dynamic_graph.sot.core import *
